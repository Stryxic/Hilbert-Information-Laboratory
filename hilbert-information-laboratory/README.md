# Hilbert Information Laboratory

This repository accompanies the PhD thesis *“Detecting Disinformation Campaigns in Social Media: Modelling Disinformation, Authorship, and Trust through Computational Chains of Trust.”*

It provides the computational environment, simulation framework, and ethically collected data used to study information integrity, provenance, and trust using the **Hilbert Information Laboratory (HIL)**.

## Repository Contents

- `data/` – Collected corpora, logs, results, and metadata
- `src/` – Core simulation and analysis code
- `notebooks/` – Interactive Jupyter notebooks for analysis
- `docs/` – Ethics statements, architecture diagrams, and thesis references
